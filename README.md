# Go simple web example

Simple [Golang](https://golang.org/) web example application, based on this [tutorial](https://www.sohamkamani.com/blog/2017/09/13/how-to-build-a-web-application-in-golang/).

## Requires

* [Go](https://golang.org/)

## Run

```
go run main.go
```

## Test

```
go test ./...
```
